Interactive Cheat Sheet — Lesson: Publishing on YouTube

Key Publishing Rules

Uploading a Video

  • Camera + plus → Upload video
  • Thumbnail — 1280×720 px important
  • Visibility: public / unlisted / private
  • Schedule — choose the premiere date and time

PRO Description

  • Line 1 — keyword + what the video is about
  • Line 2 — hook / benefit for the viewer
  • Chapters — min. 3, starts at 0:00
  • CTA — subscribe / comment / link

End Screen

  • Last 20 seconds of the video
  • Elements: Video + Subscribe
  • Video must be at least 25 sec.
  • Editor → End screen

Info Cards

  • Can appear at any moment
  • Types: video, playlist, channel
  • Set the time on the timeline
  • Details → Cards

Subtitles

  • Subtitles tab in Studio
  • Upload a .sbv / .srt file
  • Or the manual editor in YT
  • Add a second language for reach bonus

Pinned Comment

  • Add a comment under the video
  • Three dots → Pin
  • CTA or a question for viewers
  • Visible at the very top

Playlist

  • ... under the video → Save
  • Create a new playlist
  • Add a section to your channel page
  • Customize channel → Layout

SEO Tags

  • 5–8 tags per video
  • 1st tag = exact channel title
  • General + specific words
  • Invisible to viewers, read by the algorithm
