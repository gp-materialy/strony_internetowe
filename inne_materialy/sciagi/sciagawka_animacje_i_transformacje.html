Ściągawka CSS - animacje i transformacje

Po lewej widzisz efekt, po prawej kod, który możesz skopiować.

Najedź myszką na pomarańczowe pole w sekcji Przejścia, żeby zobaczyć efekt na żywo. Kliknij Kopiuj, aby przenieść kod do swojego projektu.

Animacje - animation i @keyframes

tu
ciągły obrót 
Animacja obrotu w pętlianimation: spin 2s linear infinite;

@keyframes spin {
	from { transform: rotate(0deg); }
	to { transform: rotate(360deg); }
}
tu
zmiana koloru w pętli 
Animacja koloru tłaanimation: flash 1.5s infinite;

@keyframes flash {
	0%   { background: red; }
	50%  { background: blue; }
	100% { background: red; }
}
