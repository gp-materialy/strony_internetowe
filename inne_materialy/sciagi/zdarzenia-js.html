Laboratorium zdarzeń

Zdarzenie (event) to sygnał, że coś się stało — klik, naciśnięcie klawisza, wysłanie formularza. Ta strona jest cała oprzyrządowana: każda Twoja interakcja trafia na żywo do monitora po lewej. Wypróbuj 12 najczęstszych zdarzeń i obserwuj, jak się wyzwalają.

Monitor zdarzeń

razem: 0
  • Nasłuchuję… wykonaj dowolną akcję poniżej.
click

Pojedyncze kliknięcie elementu — najczęstsze zdarzenie w sieci.

kliknięć: 0
btn.addEventListener('click', () => { … });
dblclick

Dwa szybkie kliknięcia pod rząd na tym samym elemencie.

czekam na podwójne kliknięcie…
box.addEventListener('dblclick', () => { … });
keydown

Klawisz został wciśnięty (jeszcze nie puszczony).

wciśnięto: —
input.addEventListener('keydown', e => e.key);
keyup

Klawisz został puszczony — fizyczne zwolnienie przycisku.

puszczono: —
input.addEventListener('keyup', e => e.key);
change

Wartość pola zmieniła się i pole straciło aktywność (lub wybrano opcję).

jeszcze nic nie wybrano
sel.addEventListener('change', e => e.target.value);
input

Reaguje przy każdym znaku — w trakcie pisania, na bieżąco.

znaków: 0
input.addEventListener('input', e => e.target.value);
focus

Element stał się aktywny (np. po kliknięciu lub klawiszu Tab).

nieaktywne
input.addEventListener('focus', () => { … });
blur

Element stracił aktywność — kliknięto gdzie indziej.

aktywuj i opuść pole…
input.addEventListener('blur', () => { … });
mouseover

Kursor wjechał na obszar elementu.

Najedź kursorem tutaj
box.addEventListener('mouseover', () => { … });
mouseout

Kursor opuścił obszar elementu.

Najedź, a potem zjedź kursorem
box.addEventListener('mouseout', () => { … });
submit

Formularz został wysłany. preventDefault() blokuje przeładowanie strony.

formularz nie został jeszcze wysłany
form.addEventListener('submit', e => { e.preventDefault(); // nie przeładowuj });
load

Cała strona (z obrazkami i zasobami) skończyła się ładować. To zdarzenie wyzwoliło się raz, przy starcie.

ładowanie…
window.addEventListener('load', () => { … });
Wskazówka: w monitorze widzisz typ zdarzenia, jego cel (e.target) i czas wyzwolenia. Każde z 12 zdarzeń podłączono przez addEventListener(typ, funkcja).
